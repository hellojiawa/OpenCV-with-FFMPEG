## OpenCV with FFMPEG

OpenCV版本：V3.1.0  
FFMPEG版本：3.4.1  


OpenCV在Android平台上不能使用VideoCapture，找遍互联网也没找到一个能用的版本，于是花了几天时间踩了无数坑编译了一个。
本项目需要和FFMPEG一起使用，对应的FFMPEG的so文件可以[点击这里](https://github.com/hellojiawa/FFMPEG-for-android)下载。
